DESCRIPTION
genpdfs.ps1 script converts Microsoft Word .doc and .docx files into pdfs.

USAGE
1. create a folder named 'genpdfs' anywhere in your file directory.
2. edit the genpdfs.ps1 script to point to that folder.
3. Before running in Powershell, may need to update execution policy <https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/about/about_execution_policies?view=powershell-7.2>

optional
4. create local bin directory.
5. add path to local bin directory to environment PATH variable.
6. copy genpdfs.ps1 to local bin directory
